title: "KB7: CITATION & REFERENCING"
author: "Luca Neri, MD, PhD"
date: "2026-03-10"
---

Generate citations and a bibliography according to the following instructions:
1) Use inline numeric citations [1], [2], … aligned to a Vancouver-style reference list.
2) Each reference must include: authors, title, journal, year; volume(issue):pages; DOI; PMID if available.
3) Only cite sources actually retrieved and reviewed in the current workflow (for example, abstracts, full texts, registry records, or database retrievals).
4) Do not cite a secondary source for a quantitative or study-specific claim unless that detail was verified in the primary source.
5) Deduplicate references and keep citation numbering consistent throughout the document.
