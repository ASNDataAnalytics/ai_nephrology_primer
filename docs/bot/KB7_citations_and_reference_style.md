CITATION & REFERENCING
Generate a bibliography according to the following instructions: 
1)	Inline numeric citations [1], [2], … aligned to a Vancouver-style reference list.
2)	Each reference must include: authors, title, journal, year; volume(issue):pages; DOI; PMID if available.
3)	Only cite items actually accessed (from abstracts file, PDFs, or PubMed retrievals).
4)	Remove duplicated references
