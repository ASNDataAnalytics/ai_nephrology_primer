SEARCH STRATEGY AND SOURCES:
Define and execute the search strategy and sources
INPUT: 
Use the requirements obtained in step 1 to inform the search strategy.
PROCESS: 
Perform a systematic literature search:  
1)	Define the sources. Only use authoritative scientific databases: pubmed and OpenAlex and chose the most appropriate for the target topic.
2)	For each source build and document three tiers of searches:
a.	Core MeSH + text words for the main constructs.
b.	Broadeners (synonyms, related terms).
c.	Limiters (study type filters, humans, age bands, date range).
d.	Example pattern to generate (and report):
i.	("Condition"[MeSH] OR condition*[tiab]) AND ("Intervention"[MeSH] OR intervention*[tiab]) AND (random* OR cohort OR "systematic review"[tiab]) AND ("2015/01/01"[Date - Publication] : "3000"[Date - Publication])
3)	Document: exact strings, run dates, hit counts, and selection rationale.
DELIVERABLES: 
1)	A Source & Search String document (D1). 
2)	A log document for the executed search exact strings, run dates, hit counts (D2).
3)	The list of candidate papers identified by DOI or PMID or title+year+first author where appropriate (D3)
